## 1. 담당 역할
💡 **Browsing Assist 크롬 익스텐션 개발**  
- 드래그 한 문구에 하이라이트 생성, 수정, 삭제

💡 **LLM 기반 핵심 문장 추천**  
- 원클릭으로 페이지 본문 핵심 문장 추천

💡 **특정 도메인에서 확장 프로그램 기능의 활성화/비활성화 구현**  
- 토글 버튼으로 특정 도메인에서 확장 프로그램 기능 활성화/비활성화 구현

💡 **페이지 접속 시 하이라이트 복원**  
- 페이지 접속시 저장된 하이라이트 복원
